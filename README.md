# Portofoliu Programare Orientată pe Obiecte (OOP) în C++

Acest repository conține o colecție a proiectelor mele C++, demonstrând diverse concepte de Programare Orientată pe Obiecte (OOP), structuri de date și algoritmi. Proiectele sunt organizate în două categorii principale: aplicații OOP complete și exemple algoritmice mai mici.

##  Structura Proiectului

-   **`OOP Portofolio/`**: Conține 21 de aplicații C++ complete care implementează principii OOP.
-   **`SmallProjects/`**: Conține 12 scripturi focalizate care demonstrează algoritmi specifici și mecanisme C++.

---

##  Proiecte Mici (`SmallProjects/`)

Exemple concise care se concentrează pe algoritmi specifici, manipularea datelor și caracteristici ale limbajului C++.

| Nume Proiect | Descriere |
| :--- | :--- |
| **GuestListManager** | Listă de invitați bazată pe consolă folosind manipulare simplă de string-uri. |
| **PolymorphismDemo** | Demonstrație funcții virtuale folosind clasele Animal/Pisică/Câine. |
| **PolymorphismDemo2** | Exemple extinse de polimorfism explorând destructori virtuali. |
| **SimpleStringWrapper** | Implementare de bază a unui wrapper pentru clasa string. |
| **StringInputExperiment** | Experimente cu `cin`, `getline` și manipularea input-ului de tip string. |
| **VectorAverageReplacement** | Algoritm pentru înlocuirea numerelor pare dintr-un vector cu media numerelor impare. |
| **VectorDuplicateRemoval** | Algoritm eficient pentru eliminarea valorilor duplicate dintr-un vector de întregi. |
| **VectorElementDeletion** | Logică pentru ștergerea ultimelor `k` apariții ale unei valori specifice dintr-un vector. |
| **VectorFrequencyFilter** | Filtrează elementele vectorului pe baza frecvenței lor de apariție. |
| **VectorParityProcessing** | Procesează elementele vectorului (dublare/eliminare) pe baza parității (impar/par). |
| **VectorSpecificDeletion** | Implementare alternativă pentru ștergerea elementelor specifice în vectori. |
| **VectorTransformations** | Logică complexă pentru transformarea elementelor vectorului pe baza unor reguli condiționale. |
